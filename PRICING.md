# Suggested Commercial Packaging

This file is a product plan, not a payment system. Payments should go through accounts owned by the project owner, such as Gumroad, Lemon Squeezy, Stripe Payment Links, GitHub Sponsors, or a private GitHub repository.

## Free

Price: $0

Includes:

- Codex skill workflow.
- Report structure reference.
- Valuation method reference.
- Disclaimer and verification reference.
- Basic outline generator.
- Input validator.

Goal: Build trust, attract users, and demonstrate the workflow.

## Starter Pro Pack

Suggested price: $29-$49 one-time.

Includes:

- Editable Word research report template.
- Excel valuation model starter template.
- PowerPoint stock pitch template.
- Sample completed research package.
- Extra prompts for initiation reports, earnings previews, and valuation memos.

## Sector Pack

Suggested price: $49-$99 per sector.

Includes:

- Sector-specific KPI checklist.
- Sector valuation norms.
- Peer comparison framework.
- Industry driver library.
- Sector-tailored report outline.

Suggested first sectors:

- Banks and financials.
- Consumer staples.
- Technology and software.
- Industrials.
- Real estate.

## Team Pack

Suggested price: $199-$499 one-time or annual license.

Includes:

- Internal review checklist.
- Branded report template.
- Team style guide.
- Reusable source audit workflow.
- Custom installation notes.

## Delivery Options

- Gumroad or Lemon Squeezy download package.
- Stripe Payment Link plus manual delivery.
- Private GitHub repository access.
- GitHub Sponsors for ongoing support.

## Honest Sales Copy

Good positioning:

> A workflow accelerator for drafting equity research outlines, valuation plans, and source checklists from your own materials.

Avoid:

> This tool tells you which stocks to buy.

> Guaranteed alpha from AI research.

> Automated investment advice.
